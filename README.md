# Bento Boxed 🍱
A secure Java and Spring Boot FHIR API.

Tech stack: `Java` `Spring Boot` `Spring Security` `OAuth 2.0` `PostgreSQL` `Docker` `Postman`
- Built a secure backend microservice using Java and Spring Boot to map legacy clinical datasets into compliant FHIR resource endpoints
- Implemented OAuth 2.0 and SMART on FHIR protocols via Spring Security to establish strict token authentication for patient records
- Designed a relational PostgreSQL schema to manage complex medical metadata with indexing to maintain low-latency read performance
- Containerized the application using Docker and constructed rigorous end-to-end API test suites inside Postman to validate edge cases
